NGSS Alignment Crosswalk: IoT & Sensor Activities

How micro:bit sensor activities map to Next Generation Science Standards Performance Expectations.

How to use this crosswalk: Find your grade band and discipline, then pick a sensor activity that maps to a PE you're already teaching. The micro:bit becomes the data collection tool for a standard you already cover.
