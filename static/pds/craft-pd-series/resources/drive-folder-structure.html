Setup Instructions

Templates: For Google Doc templates (lesson template, CtM template, IoT template), set sharing to "Anyone with the link can view" and instruct participants to go to File → Make a copy. This prevents accidental edits to the master.
Collaboration Docs: Each workshop's group workspace should have "Anyone with the link can edit." Create a table of contents at the top with sections for each breakout group.
MakeCode .hex files: Export from MakeCode by clicking the gear icon → "Save Project" → download .hex. Participants can import these via "Import" in MakeCode.
Digital Toolkit PDFs: To create the per-session toolkit PDF, print each HTML resource page to PDF (Ctrl+P → Save as PDF) and combine them using any PDF merger. Each workshop's toolkit should include all resources from that workshop's folder.
