Common Pitfalls Across All Platforms

They all hallucinate standards. Every platform sometimes invents NGSS codes that don't exist or maps to the wrong grade band. Always verify standard numbers against the actual NGSS framework.
They all struggle with math precision. Simple arithmetic is usually fine, but multi-step calculations (especially unit conversions, significant figures, and statistics) frequently have errors. Treat all math output as a draft.
They all sound confident when wrong. None will spontaneously say "I'm not sure about this." The burden of verification is always on you. This is why we teach Check the Machine.
Same prompt, different output every time. LLMs are non-deterministic. Running the exact same prompt twice may give different results. This is a feature, not a bug — but it means you can't assume one good output means the next will be identical.

Quick Decision Flowchart

Do I need to generate or run code? → ChatGPT
Do I need to follow a complex template? → Claude
Do I need current web info? → Gemini or ChatGPT
Do I need Google Workspace integration? → Gemini
Do I need to analyze uploaded files? → Any (all support file upload)
Not sure? → Try all three with the same prompt and compare. That's the whole point of Workshop 1!
