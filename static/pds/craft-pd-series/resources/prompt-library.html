Prompt Anatomy

Every effective prompt has four components. The more specific you are, the better the output.

ROLEWho the AI should be — "You are an experienced 9th grade physics teacher..."
TASKWhat you want — "Create a 5-day inquiry-based unit on..."
CONSTRAINTSBoundaries — grade level, reading level, NGSS standards, time, materials available
FORMATWhat the output should look like — "Output as a table with columns for..."

Physics

Lesson Generation · Grades 9-12
Inquiry-Based Projectile Motion Unit
You are an experienced high school physics teacher who designs inquiry-based lessons. Create a 5-day unit on projectile motion for 9th graders aligned to NGSS HS-PS2-1 (Newton's Second Law). Requirements: - Each day should take 50 minutes - Include a hands-on lab using only common classroom materials (rulers, balls, tape, stopwatches) - Differentiate for 3 levels: struggling (8th-grade reading), on-level, advanced (calculus-ready) - Include 2 formative assessment checkpoints - Day 5 should be a student-designed investigation Output as a table: Day | Objective | Activity | Materials | Assessment | Differentiation Notes
Why it works: Specifies the standard, time constraints, material limitations, and differentiation tiers. The table format makes the output immediately usable.
Problem Set · Grades 6-8
Force & Motion Practice Problems
You are a middle school science teacher creating practice problems. Generate 12 problems about forces and motion for 7th graders aligned to NGSS MS-PS2-2 (force = mass × acceleration). Structure: - Problems 1-4: Conceptual (no calculation, predict direction/magnitude) - Problems 5-8: Single-step calculation (given F, m, find a or vice versa) - Problems 9-12: Multi-step with real-world context (sports, vehicles, animals) For each problem include: the problem text, the answer, a 1-sentence explanation of the key concept, and the common misconception this problem targets.

Biology / Life Science

Lab Design · Grades 6-8
Photosynthesis Investigation
You are a middle school life science teacher designing a 3-day lab sequence on photosynthesis aligned to NGSS MS-LS1-6 (role of photosynthesis in cycling matter and flow of energy). Day 1: Students design an investigation to test one variable affecting photosynthesis rate (light intensity, CO2 levels, or water availability). They must write a hypothesis and procedure. Day 2: Students conduct the investigation and collect data. Day 3: Students analyze data, create a graph, and write a CER (Claim-Evidence-Reasoning) conclusion. For each day, provide: learning objective, teacher setup instructions, student handout text (written at 6th grade reading level), safety considerations, and expected results with common errors students make.
Reading Companion · Grades 9-12
Cell Division Misconception Buster
You are a biology teacher who specializes in identifying and correcting student misconceptions. Create a "Misconception vs. Reality" handout about cell division (mitosis and meiosis) for 10th graders aligned to NGSS HS-LS3-2. Include exactly 8 misconception pairs: - State the common misconception students hold - Explain why students develop this misconception - Provide the correct scientific explanation using precise vocabulary - Give a simple analogy that helps students remember the correct concept Format each entry clearly with headers. Use language appropriate for 10th graders. Do NOT oversimplify to the point of inaccuracy.

Mathematics

Personalized Tutor · Grades 3-5
Fraction Word Problems (Scaffolded)
You are a patient, encouraging elementary math tutor. Create 10 word problems about adding and subtracting fractions with unlike denominators for 4th graders. Requirements: - Use real-world contexts students care about (cooking, sports, games, animals) - Problems 1-3: Visual support — include a description of a fraction bar model for each - Problems 4-7: Guided — provide the first step, students complete the rest - Problems 8-10: Independent — no scaffolding - All fractions should use denominators ≤ 12 - Include an answer key with step-by-step solutions - Flag which problems align to CCSS 4.NF.B.3a vs 4.NF.B.3b
Assessment · Grades 9-12
Quadratic Functions Performance Task
You are a high school math teacher creating an authentic performance task about quadratic functions for Algebra 2 students. The task should involve a real engineering scenario (bridge design, projectile trajectory, or parabolic antenna) where students must: 1. Write a quadratic equation from given constraints 2. Find the vertex and interpret it in context 3. Determine the x-intercepts and interpret them 4. Graph the function with labels 5. Write a recommendation based on their analysis Include: the complete task prompt (1 page), a scoring rubric (4-point scale for each criterion), and sample student responses at the "proficient" and "developing" levels. Align to CCSS HSF-IF.C.7a.

Computer Science

Unplugged Activity · Grades K-2
Algorithm Storytelling
You are a K-2 teacher who integrates computing concepts into literacy lessons. Design a 30-minute unplugged activity that teaches the concept of "algorithms" (step-by-step instructions) through storytelling. Requirements: - Use a familiar fairy tale (Three Little Pigs, Goldilocks, or Little Red Riding Hood) - Students create an "algorithm" for the main character using picture cards (arrows, stop signs, decision diamonds) - Include exactly 6 picture card descriptions I can print - Activity should work for a class of 24 with mixed reading levels - Provide teacher script for the 5-minute introduction - Include 3 extension questions for early finishers - Align to CSTA K-2 1A-AP-08 (model daily processes by creating and following algorithms)
Debugging Exercise · Grades 6-8
micro:bit Debugging Challenge
You are a middle school CS teacher. Create a set of 4 MakeCode (block code) debugging challenges for the BBC micro:bit. Each challenge should have code that ALMOST works but has one logical error. For each challenge provide: 1. What the code is supposed to do (in plain English) 2. The block code description (describe each block in order since I'll recreate it in MakeCode) 3. The specific bug and where it is 4. The fix 5. A hint to give students who are stuck (without revealing the answer) Difficulty progression: Challenge 1 uses only basic blocks, Challenge 4 uses variables and conditionals. All should involve the micro:bit's onboard sensors (buttons, accelerometer, temperature, light).

General / Cross-Subject

Lesson Alignment · Any Grade
NGSS Standards Mapper
I am a [GRADE] [SUBJECT] teacher. I want to teach a lesson about [TOPIC]. Identify the 3 most relevant NGSS performance expectations for this topic at this grade band. For each standard: 1. State the full performance expectation code and text 2. Identify the Science & Engineering Practice it emphasizes 3. Identify the Crosscutting Concept 4. Identify the Disciplinary Core Idea 5. Suggest one specific classroom activity (30-45 min) that addresses this standard 6. Note one common assessment mistake teachers make when assessing this standard Replace the bracketed terms with your specifics before using this prompt.
Discussion Starter · Grades 6-12
Socratic Seminar Question Generator
You are a skilled facilitator of student-centered discussions. Generate 10 Socratic seminar questions about [TOPIC] for [GRADE] students. Structure the questions in three tiers: - Questions 1-3: Opening (accessible, opinion-based, gets everyone talking) - Questions 4-7: Core (analytical, requires evidence, connects to key concepts) - Questions 8-10: Closing (synthesis, real-world connection, "so what?" questions) For each question, include: - The question text - The type (opening/core/closing) - A follow-up probe if students give a surface-level answer - The key concept or misconception this question targets Questions should be open-ended (no yes/no) and should not have a single correct answer.

Differentiation Prompts

Differentiation · Any Lesson
Three-Tier Differentiation Converter
I have an existing lesson about [TOPIC] for [GRADE] students. Here is the core activity: [PASTE YOUR ACTIVITY HERE] Create three differentiated versions of this activity: TIER 1 (Approaching): Simplify vocabulary to [TARGET] reading level. Add sentence starters. Reduce the number of steps. Provide a partially completed example. Add visual supports. TIER 2 (On-Level): Keep the activity as-is but add one extension question that requires application to a new context. TIER 3 (Advanced): Add complexity by requiring students to [evaluate/create/compare]. Include a challenge that connects to a related concept they haven't studied yet. For each tier, output the complete modified activity text I can hand directly to students.

Admin Time-Savers

Communication · Parents
Weekly STEM Newsletter
You are a friendly, professional [GRADE] [SUBJECT] teacher writing a weekly parent newsletter. This week we worked on [TOPIC]. Write a 200-word newsletter that includes: 1. A brief summary of what students learned (no jargon, parent-friendly) 2. One conversation starter parents can use at dinner ("Ask your child about...") 3. One simple at-home extension activity using household materials 4. A vocabulary word of the week with a kid-friendly definition 5. A preview of next week's topic Tone: warm, encouraging, makes parents feel included without overwhelming them. Do NOT use educational buzzwords.
Rubric · Any Assignment
Single-Point Rubric Generator
Create a single-point rubric for the following assignment: [PASTE YOUR ASSIGNMENT DESCRIPTION] The rubric should have: - 4-6 criteria rows (identify the most important skills being assessed) - Three columns: "Concerns" (blank, for teacher notes) | "Criteria" (the proficient-level description) | "Advanced" (blank, for teacher notes) - Each criterion should be specific, observable, and measurable - Include a row for the relevant NGSS or content standard alignment Output as a clean table I can paste into a Google Doc.
