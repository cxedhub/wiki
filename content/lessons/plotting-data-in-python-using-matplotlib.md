# Lesson Plan
## Overview
CSV files can be read using pandas read_csv method. Using this method you can easily use the dataframes created to plot the data. For today’s exercise we will be using the recorded flight time for the x-axis of our line charts. Students will record flight data from their drones and plot the data against time using Python.
## ASSESSMENT PRE/POST-TEST
What type of chart/graph would you use to show a data series over time?
How does Python use pandas to show a data series?
What library/module will you import into your python program to chart data?
## OBJECTIVES
1. Read a csv file with Python.
2: Use the Pandas library to read the data into a dataframe.
3: Plot data from the dataframes using Matplotlib.

## CATCH/HOOK
Prove how high you can fly with MVHS technology using our amazing fleet of drones! Students will test the limits of their drones and compare the resulting data.

## ACTIVITY INSTRUCTIONS
1.	Have your drone complete a mission
2.	Collect the data from the drone
3.	Create a program in Python called DronePlot.py. Make sure to put your name in the title of the chart.
4.	Create a plot of your battery voltage over time.
5.	Save the image as a pdf file called battery.pdf
6.	Create a plot of your flight height over time.
7.	Save the image as a pdf file called height.pdf
8.	Create a  plot of your flight Air Pressure over time.
9.	Save the image as pdf  file called pressure.pdf
10.	Submit all three charts and your code to this assignment.
11.	Good Luck Pilots!
