# Lesson Plan
## Overview
This presentation will introduce students to the foundational concepts of the Python programming language. After a brief introduction on what Python is the students are introduced to variables, language syntax, data types, inputs and outputs, If/Else statements and Loops. The presentation is followed up with a Kahoot! game (TBD) based on the content so the students can recall what they have learned.
## ASSESSMENT PRE/POST-TEST
The post test will be replaced with a Kahoot game that is still to be developed.
## OBJECTIVES
Understand concepts like variables, language syntax, data types, inputs and outputs, If/Else statements and Loops as they relate to the Python programming language.

## CATCH/HOOK
What is your favorite app to use? Have you ever thought about how it works?

## ACTIVITY INSTRUCTIONS
Using the power point presentation begin introducing the students to the Python programming language. With each slide try to ask questions and check for understanding along the way. Try to engage in a discussion wherever possible. Have some helpful coding analogies ready to go to assist in explaining how things work. For example, a variable is like a bucket that you put things in etc.

## REVIEW
To wrap things up load the Kahoot game and have the students join from their computers. For fun you can prepare a prize at the end for the winners and/or participants.
