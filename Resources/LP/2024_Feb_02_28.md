# Lesson Plan
## Overview
Creating a program for their microbit 'pets'.  Their program must include noise, pictures and words/phrases
## ASSESSMENT PRE/POST-TEST
What is a robot?
## OBJECTIVES
Program a microbit to perform various tasks.


## CATCH/HOOK
Microbit pets!


## ACTIVITY INSTRUCTIONS
Program a microbit
